# Olympic Stats App

## Status: `Completed`

## Description

Olympic Stats App allows you to see last Olympic Medals Stats. You can manually sort rows using functions in headers of table. Choose between English and Polish.

## Technologies

- React(Next.js)
- Typescript
- styled-components
- MUI

## Setup

```bash
## Setup and run app
- run 'yarn'
- run 'yarn dev'

## Available commands
- dev # developing site
- build # production build
```
